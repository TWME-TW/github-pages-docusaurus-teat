# 遊玩方式

創造建築伺服器大致有兩種遊玩方式，分別為：在 **主世界** 建築，與在 **個人世界** 建築。

在主世界建築就是在第一次進入伺服器時，所進入的世界中建築。

{% content-ref url="build-together.md" %}
[build-together.md](build-together.md)
{% endcontent-ref %}

而個人世界建築則是可以創建一個主於自己的世界，你選擇公開或者是私人的世界，並在裡面自由的建築。
